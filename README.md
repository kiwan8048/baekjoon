# Baekjoon Algorithm Solutions
 
📌 백준(BOJ) 알고리즘 문제 풀이 모음입니다.  
문제는 C, Python, C++ 등 여러 언어로 풀이하고 있으며,  
**실버~골드 난이도** 위주로 실전 감각을 기르기 위해 작성하고 있습니다.

---


- 문제 번호 + 간단한 제목으로 파일명을 작성
- 난이도별로 폴더 분류 (silver / gold / etc)

---

## ✍️ 목표

- 실버/골드 문제 정리
- 코딩테스트 실전 감각 익히기
