# Baekjoon Algorithm Solutions

📌 백준(BOJ)에서 푼 알고리즘 문제들을 정리한 저장소입니다.  
모든 문제는 C 언어로 풀이했으며, 실버 ~ 골드 난이도를 위주로 풀이하고 있습니다.

---


- 문제 번호 + 간단한 제목으로 파일명을 작성
- 난이도별로 폴더 분류 (silver / gold / etc)

---

## ✅ 풀이 언어

- C (GCC 11.2 기준)
- 기본 입출력: `scanf`, `printf`
- 필요한 경우 `stdlib.h`, `string.h` 사용
